# Weather-Beaten Journal

**Weather-Beaten Journal** is a fishing companion addon for World of Warcraft — a catch log, zone guide, and collectible tracker, all narrated by Azeroth's finest anglers.

---

## Features

### Catch Log
- Automatically records every fish caught with timestamps, zones, and counts
- Searchable and sortable by recent, count, name, rarity, or unlearned
- Session stats: total catches, unique species, missed casts

### Spot Guide
- Interactive zone browser with expansion filtering
- Shows available fish per zone with catch data
- Context-sensitive NPC narration with portraits and flavor text

### Collections
- Tracks 60+ fishing achievements organized by category
- Monitors mounts, pets, toys, Underlight Angler, Garrison Fishing Shack, and Angler reputation progress
- Clickable entries open the relevant achievement or collection UI

### Narrators
- 21 fishing NPCs across all expansions provide themed narration
- Contextual quotes that change based on your current view and zone

---

## Usage

The journal integrates directly into the Professions UI as a **Weather-Beaten** tab, visible when the Fishing profession window is open.

Slash commands:

- `/wbj` — Open the journal
- `/wbj stats` — Show catch statistics
- `/wbj reset` — Wipe all saved data
- `/wbj help` — Show help text

---

## Installation

Download and extract into:

```
World of Warcraft/_retail_/Interface/AddOns/WeatherBeatenJournal
```

Restart WoW or `/reload`.
